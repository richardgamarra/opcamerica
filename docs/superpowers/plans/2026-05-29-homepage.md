# OPCAmerica Homepage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the bilingual (EN/ES) OPCAmerica homepage in Next.js 14 — 8 sections from cinematic hero to email capture, fully deployable on Vercel.

**Architecture:** Next.js 14 App Router with a `[lang]` dynamic segment handles `/en/` and `/es/` routes. Bilingual strings live in `lib/i18n/en.ts` and `lib/i18n/es.ts`. No CMS in this phase — all content is static. `middleware.ts` redirects `/` to `/en/` on first visit.

**Tech Stack:** Next.js 14 (App Router), Tailwind CSS, TypeScript, Vercel

- [ ] **Step 1: Create middleware.ts**

```typescript
import { NextResponse } from "next/server";
import type { NextRequest } from "next/server";

const VALID_LANGS = ["en", "es"];

export function middleware(request: NextRequest) {
  const { pathname } = request.nextUrl;

  // Skip non-page paths
  if (
    pathname.startsWith("/_next") ||
    pathname.startsWith("/api") ||
    pathname.startsWith("/images") ||
    pathname.includes(".")
  ) {
    return NextResponse.next();
  }

  // Already has valid lang prefix
  const langSegment = pathname.split("/")[1];
  if (VALID_LANGS.includes(langSegment)) {
    return NextResponse.next();
  }

  // Check cookie for saved preference
  const cookieLang = request.cookies.get("lang")?.value;
  const lang = cookieLang && VALID_LANGS.includes(cookieLang) ? cookieLang : "en";

  return NextResponse.redirect(new URL(`/${lang}${pathname}`, request.url));
}

export const config = {
  matcher: ["/((?!_next|api|images|favicon.ico).*)"],
};
```

- [ ] **Step 1: Create components/layout/Nav.tsx**

```typescript
"use client";

import Link from "next/link";
import { useRouter } from "next/navigation";
import type { Translations, Lang } from "@/lib/i18n";

interface NavProps {
  lang: Lang;
  t: Translations["nav"];
}

export function Nav({ lang, t }: NavProps) {
  const router = useRouter();
  const otherLang = lang === "en" ? "es" : "en";

  function switchLang() {
    document.cookie = `lang=${otherLang};path=/;max-age=31536000`;
    router.push(`/${otherLang}/`);
  }

  return (
    <nav className="absolute top-0 left-0 right-0 z-10 flex justify-between items-center px-12 py-5"
         style={{ background: "linear-gradient(to bottom, rgba(5,10,20,0.9) 0%, transparent 100%)" }}>
      <Link href={`/${lang}/`} className="font-sans font-black text-xl tracking-tight text-white">
        OPC<span className="text-opc-orange">America</span>
      </Link>

      <div className="hidden md:flex gap-8 font-sans text-sm text-white/70">
        {(["launchpad","tutorials","aiTools","community","consulting"] as const).map((key) => (
          <Link key={key} href={`/${lang}/#${key}`} className="hover:text-white transition-colors">
            {t[key]}
          </Link>
        ))}
      </div>

      <div className="flex gap-3 items-center font-sans text-sm">
        <button
          onClick={switchLang}
          className="border border-white/30 px-4 py-1.5 rounded-full text-white/75 hover:text-white transition-colors text-xs"
        >
          🌐 {t.langToggle}
        </button>
        <Link href={`/${lang}/signin`} className="text-white/70 hover:text-white">
          {t.signIn}
        </Link>
        <Link
          href={`/${lang}/join`}
          className="bg-opc-orange text-white px-5 py-2 rounded-lg font-bold hover:bg-opc-orange/90 transition-colors"
        >
          {t.joinFree}
        </Link>
      </div>
    </nav>
  );
}
```
